# mint-classics-inventory-optimization
SQL-based inventory analysis to optimize storage facilities and sales velocity for a classic car retailer.
# Mint Classics Inventory Analysis 🏎️

## Project Overview
This project addresses the business problem of identifying which warehouse the Mint Classics Company can close to reduce costs while maintaining service.

## Key Insights
- **Warehouse B (East)** was identified as the target for closure due to an average inventory-to-sales ratio of **14.5 years**.
- **Fast-Movers**: Identified high-velocity items (like the 1960 BSA Gold Star) that require restocking, not reduction.
- **Dead Stock**: Isolated items with 20+ years of inventory that should be liquidated.

## Skills Demonstrated
- **CTEs (Common Table Expressions)** for complex logic.
- **Subqueries** for statistical filtering.
- **Data Normalization** (Annual Sales Velocity).
